CAMBIOS ENFOCADOS SOLO EN EL LAG AL SUBIR Y BAJAR
- El feed usa LazyColumn con claves y contentType estables.
- Se elimino el limite visual de 24 tarjetas; LazyColumn conserva solo las tarjetas visibles y permite carga progresiva hasta 60 elementos de datos.
- La carga adicional ahora es automatica al acercarse al final; se elimino el boton manual Cargar mas.
- Mientras el dedo esta desplazando, las miniaturas nuevas no se decodifican: muestran un marcador simple y cargan cuando el movimiento termina.
- Las miniaturas que ya estaban decodificadas permanecen visibles durante el desplazamiento.
- Solo se precargan las dos miniaturas siguientes y unicamente cuando el scroll se detiene.
- Las miniaturas del feed se solicitan a 480x270 y las de avatar a 72x72.
- La API deja de priorizar maxres y prefiere miniaturas medianas para no descargar imagenes multipixel por cada tarjeta.
- Coil usa cache de memoria, disco y red de forma explicita, sin crossfade.
- Se quitaron divisores y superficies innecesarias de cada tarjeta.
- La fila de Shorts conserva 8 elementos y usa imagenes limitadas.
- El workflow compila la variante release V22.
